SOLIDITY

Solidity is a statically-typed curly-braces programming language designed for developing smart contracts that run on the Ethereum Virtual Machine.

Get Started

Solidity is still evolving rapidly, you can participate in the language design process by contributing.

GOVERNANCE & LANGUAGE DESIGN

We welcome you to get involved and take part in out language design process.

  • Take part in the design discussions in the Github issues.
  • Join the solidity-users forum: proposals for new language features and qualities of existing aspects of the language can be discussed.
  • Take part in (feature) feedback surveys.
  • Join dedicated language design discussion calls, where one topic/issue/feature implementation is debated per call.
